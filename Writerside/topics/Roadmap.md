# Roadmap

This project is actively evolving. Highlights from the current roadmap:

## Completed

- Theme support
- Markdown and LaTeX rendering
- Template support for flashcards
- Database export/import
- Llama.cpp integration via OpenAI-compatible Spring AI
- Optional build-time dependency caching via Nexus (Maven + APT)
- SRS-based review mode and cram mode
- Tag filtering + tag cloud
- Swagger / OpenAPI support
- SQLite single-file mode

## Planned

- Lower-friction deployment options (single-container, managed DB, etc.)
- Import/export in additional flashcard formats (CSV/YAML/TOML/Anki)
- Statistics and progress tracking
- Multi-language UI
- More AI providers/models
